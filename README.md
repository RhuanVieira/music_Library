# 🎵 Michael Jackson Music Library

Uma biblioteca musical inspirada no maior artista pop de todos os tempos: Michael Jackson.  
O projeto funciona como um CRUD completo de músicas, permitindo cadastrar, visualizar, editar e remover músicas através de uma interface moderna e intuitiva.

---

## 🚀 Tecnologias utilizadas

### 💻 Front-end
- HTML5
- CSS3
- JavaScript

### ⚙️ Back-end
- Node.js
- Express.js
- Prisma ORM

### 🗄️ Banco de Dados
- MariaDB / MySQL

---

## 📚 Funcionalidades

✅ Cadastro de músicas  
✅ Listagem de músicas  
✅ Atualização de músicas  
✅ Exclusão de músicas  
✅ Integração com YouTube  
✅ Exibição de capas dos álbuns  
✅ API REST  
✅ CRUD completo
